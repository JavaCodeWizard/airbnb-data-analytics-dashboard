# Global Airbnb Performance Dashboard | Power BI

This project presents an interactive **Power BI dashboard** that analyzes Airbnb’s global marketplace performance.  
The objective of the project is to transform raw marketplace data into **meaningful insights through data visualization and analytical storytelling.**

The dashboard explores **platform growth, market concentration, pricing behavior, customer satisfaction, and user engagement patterns.**

This project was developed as part of the **Microsoft Elevate x AICTE 4-Week Power BI Virtual Internship Program.**

Due to GitHub file size limitations, the full Power BI dashboard file (.pbix) is hosted on Google Drive.

# Project Objectives

This dashboard was designed to answer important analytical questions about Airbnb’s global ecosystem:

- How has Airbnb’s listing supply evolved over time?
- Which cities dominate Airbnb’s global marketplace?
- What pricing patterns exist across property types?
- Which cities provide the best customer satisfaction ratings?
- How do customers interact with the platform through reviews?
- Are there seasonal patterns in travel demand?
- How strong are trust indicators such as host verification?

---

# Dashboard Pages

<details>
<summary><b>Page 1 – Platform Growth & Market Evolution</b></summary>

This page analyzes how Airbnb listings evolved globally and highlights major phases in the platform’s lifecycle.

### Key Insights

- Airbnb experienced **rapid expansion between 2012–2016** as the platform scaled across global cities.
- Listing growth **stabilized as the marketplace matured**.
- A noticeable **decline in 2020 reflects the impact of the COVID-19 pandemic on global travel demand.**

### Purpose

This page provides a **strategic overview of Airbnb’s growth trajectory and marketplace evolution.**

</details>

---

<details>
<summary><b>Page 2 – Market Share, Pricing & Customer Satisfaction</b></summary>

This page focuses on **city-level performance, pricing dynamics, and guest satisfaction metrics.**

### Key Analytical Questions

- Which cities dominate Airbnb’s global listings and reviews?
- Is the platform’s supply concentrated in a few markets or distributed globally?
- Which property types command the highest average price?
- Which cities consistently deliver strong customer satisfaction ratings?

### Key Insights

- **Paris, New York, and Sydney** contribute a significant share of Airbnb listings and reviews globally.
- **Hotel rooms show the highest average pricing** among property types.
- Cities such as **Mexico City and Rio de Janeiro demonstrate strong overall guest satisfaction ratings.**

</details>

---

<details>
<summary><b>Page 3 – Customer Behavior, Seasonality & Trust Signals</b></summary>

This page analyzes **how customers interact with the Airbnb platform and how demand changes over time.**

### Key Findings

- Review activity follows a **Pareto distribution**, where most guests leave only a single review.
- Cities such as **Paris and Rome experience demand peaks during April–August**, reflecting seasonal travel patterns.
- A large proportion of hosts are **fully verified**, reinforcing strong trust signals within the platform.

</details>

- **Overall Ratings View**
- **Detailed Ratings Breakdown**

This interaction allows viewers to **explore insights at different levels of analytical depth without leaving the page**, improving usability and storytelling.

# Key Analytical Insights

The analysis of Airbnb marketplace data revealed several important patterns:

- Airbnb experienced its most significant expansion phase between **2012 and 2016**, reflecting rapid global adoption.
- Major tourism destinations such as **Paris, New York, and Sydney** dominate both listings and review activity.
- **Hotel room listings command the highest average prices**, indicating premium pricing within certain accommodation categories.
- Customer engagement follows a **Pareto distribution**, where the majority of users leave only a single review.
- Travel demand demonstrates **strong seasonal trends**, particularly across European destinations during peak summer months.
- Host verification plays an important role in maintaining **trust and credibility across the platform.**
